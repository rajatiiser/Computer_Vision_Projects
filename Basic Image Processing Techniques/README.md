# Basic Image Processing Techniques using OpenCV

## Overview

This mini-project demonstrates the implementation of seven fundamental image processing techniques using OpenCV and built-in datasets from the `skimage.data` library. The objective is to understand how images can be enhanced, analyzed, and modified using classical computer vision methods without employing deep learning models.

## Techniques Implemented

1. Colorspace Conversion
2. Gaussian Blur
3. Median Blur for Denoising
4. Thresholding
5. Edge Detection
6. Drawing Operations
7. Contour Detection

## Datasets Used

The project uses built-in sample images from the `skimage.data` library:

* Coffee Image
* Rocket Image
* Moon Image
* Page Image
* Brick Image

No external datasets were downloaded or uploaded.

## Tools and Libraries

* Python
* OpenCV
* NumPy
* Matplotlib
* scikit-image

## Results

The implemented techniques successfully demonstrated:

* RGB channel separation and grayscale conversion
* Image smoothing using Gaussian filtering
* Salt-and-pepper noise removal using median filtering
* Binary and adaptive thresholding
* Edge extraction using Sobel and Canny operators
* Image annotation using geometric shapes and text
* Object boundary extraction using contour detection

## Conclusion

The project demonstrates how classical image processing techniques can be used for image enhancement, feature extraction, noise reduction, and object boundary detection. These techniques provide the foundation for more advanced computer vision applications.
