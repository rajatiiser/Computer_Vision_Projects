# Mini-Project 3: Object Detection using YOLOv8

## Overview

This project demonstrates object detection using the YOLOv8 deep learning architecture. The objective is to compare the detection performance of two pretrained YOLOv8 variants, YOLOv8n (Nano) and YOLOv8s (Small), on the same input image.

The experiment is performed on the coffee image obtained from the skimage dataset. The outputs generated by both models are analysed to study the effect of model size on object detection performance.

---

## Objectives

* Implement object detection using YOLOv8.
* Compare YOLOv8n and YOLOv8s pretrained models.
* Visualize and analyse detection outputs.
* Study the influence of model complexity on detection performance.

---

## Dataset

The project uses the coffee image provided by the skimage dataset.

Dataset Source:

* skimage.data.coffee()

No external dataset downloads were required.

---

## Models Used

### YOLOv8n (Nano)

* Lightweight architecture
* Faster inference
* Lower computational cost

### YOLOv8s (Small)

* Larger architecture
* Improved feature extraction
* Better detection accuracy

---

## Results

### YOLOv8n

Detected Objects:

* Cup
* Dining Table

### YOLOv8s

Detected Objects:

* Cup
* Dining Table
* Spoon

The YOLOv8s model produced higher confidence scores and detected an additional object compared to YOLOv8n.

---

## Key Observation

Although a plate is visible below the cup, it was not detected by either model because the pretrained YOLOv8 models are trained on the COCO dataset, which does not include "plate" as a target object category.
